# Smart Finance Tracker 💰

## Overview

Smart Finance Tracker is a full-stack web application designed to help users efficiently manage personal finances. The platform allows users to track income, monitor expenses, visualize spending patterns, and gain insights through an interactive dashboard.

---

## Features

* Track income and expenses
* Manage transaction history
* Interactive dashboard and analytics
* Financial data visualization
* User-friendly responsive interface
* Real-time financial insights
* Organized expense management

---

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate into the project:

```bash
cd finance-tracker
```

Install dependencies:

```bash
cd client
npm install

cd ../server
npm install
```

Run the application:

```bash
# Backend
cd server
npm start

# Frontend
cd client
npm run dev
```
